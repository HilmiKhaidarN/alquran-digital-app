# AL-QURAN DIGITAL APP

A HYBRID WEB AND ANDROID APPLICATION FOR READING THE HOLY QURAN WITH COMPLETE FEATURES INCLUDING AUDIO PLAYER WITH MULTIPLE RECITERS, BOOKMARK SYSTEM, AUTOMATIC PRAYER TIMES BASED ON GPS LOCATION, ADHAN NOTIFICATIONS WITH CUSTOMIZABLE SOUNDS, ARABIC FONT SETTINGS WITH MULTIPLE FONT FAMILIES, DARK MODE, COPY AND SHARE VERSES, AND OFFLINE SUPPORT. BUILT WITH HTML5, CSS3, JAVASCRIPT, AND CAPACITOR FOR CROSS-PLATFORM COMPATIBILITY.
